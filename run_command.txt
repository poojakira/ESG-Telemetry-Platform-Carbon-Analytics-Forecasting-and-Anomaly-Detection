# EcoTrack-Enterprise: Quick Start Guide

Follow these commands to get the "Supreme Production" version running on your local machine.

## 1. Environment Setup
It is recommended to use a virtual environment.

```powershell
# Create and activate virtual environment
python -m venv venv
.\venv\Scripts\activate

# Install all dependencies
pip install -r backend/requirements.txt
pip install -r frontend/requirements.txt
```

## 2. Model Training (Mandatory for AI features)
Run this once to generate the refined weight tensors for prediction and security.

```powershell
# Run from the project root
$env:PYTHONPATH="backend"; python backend/app/ml/train.py
```

## 3. Launch Services
Open two terminal windows to run the backend and frontend simultaneously.

### Terminal A: FastAPI Backend
```powershell
cd backend
uvicorn app.main:app --port 8000 --host 127.0.0.1 --reload
```

### Terminal B: Streamlit Dashboard
```powershell
cd frontend
streamlit run dashboard.py --server.port 8501 --server.address 127.0.0.1
```

---

## Alternative: Docker Deployment
If you have Docker installed, you can launch the entire stack with a single command:

```bash
docker-compose up --build
```

**Access Points:**
- **Dashboard**: http://127.0.0.1:8501
- **API Health**: http://127.0.0.1:8000/health
